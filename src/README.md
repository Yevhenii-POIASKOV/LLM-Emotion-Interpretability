# ⚙️ Вихідний код (Source Code)

Це "серце" проекту. Кожен файл відповідає за свій етап дослідження.

## Взаємодія модулів

1. **`data_loader.py`** ➔ Готує батчі токенів та мітки емоцій.
2. **`model_inspector.py`** ➔ Приймає токени, проганяє через модель і дістає "сирі" активації нейронів через Hooks.
3. **`analyzer.py`** ➔ Обробляє сирі числа, рахує середнє значення для Positive/Negative та обчислює різницю (Delta).
4. **`visualizer.py`** ➔ Бере дані з аналізатора і малює графіки.
5. **`clustering.py`** ➔ Виконує фінальну бінарну кластеризацію для ідентифікації конкретних індексів нейронів.

---
**Для розробників:** При зміні інтерфейсу функцій обов'язково повідомляйте всю команду!
